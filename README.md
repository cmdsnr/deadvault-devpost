# DeadVault Frontend

DeadVault is a digital legacy platform built for LCS Hack to the Future.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Firebase Auth
- Firestore
- Firebase Admin
- Google Cloud Storage
- Resend

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
